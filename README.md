# Log4Me Tistory Skin

`log4me`의 미니멀 다크 블로그 스타일을 티스토리 치환자 기반 스킨으로 옮긴 버전입니다.

## 포함 파일

- `index.xml`: 스킨 정보, 기본 설정, 커스텀 옵션
- `skin.html`: 티스토리 치환자 기반 템플릿
- `style.css`: Log4Me 스타일과 티스토리 댓글/방명록 보정 스타일
- `preview.gif`: 티스토리 스킨 등록용 미리보기 파일
- `images/script.js`: 아주 작은 초기화 스크립트

## 적용 방법

1. 티스토리 관리자에서 `꾸미기 > 스킨 변경 > 스킨 등록`으로 이동합니다.
2. `index.xml`, `skin.html`, `style.css`, `preview.gif`, `images/script.js`를 업로드합니다.
3. 저장 후 보관함에서 `Log4Me Tistory` 스킨을 적용합니다.
4. `꾸미기 > 스킨 편집`에서 목록 제목, 블로그 설명, 강조 색상을 조정합니다.

## 참고

티스토리는 업로드 방식에 따라 폴더를 그대로 선택하지 못할 수 있습니다. 그 경우 `images/script.js` 파일을 직접 선택해서 올리면 티스토리에서 `images` 경로로 배치합니다.
